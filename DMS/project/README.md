


# Driver Monitoring System (DMS)

Система мониторинга состояния водителя на основе компьютерного зрения. Анализирует видеопоток в реальном времени, детектирует признаки усталости, отвлечения внимания и опасного поведения.

## Возможности

- **Детекция лица и глаз** — 478 точек лицевых ориентиров (MediaPipe Face Landmarker)
- **Анализ усталости** — PERCLOS, частота и длительность морганий
- **Оценка позы головы** — углы поворота, наклона, кивков (pitch, yaw, roll)
- **Определение направления взгляда** — классификация зон внимания (дорога, панель, зеркала, телефон)
- **Детекция зевания** — анализ Mouth Aspect Ratio (MAR)
- **Система оповещений** — визуальные и консольные предупреждения с подавлением повторов
- **Логирование событий** — сохранение истории в JSON, автосохранение скриншотов при критических событиях

## Установка

### Требования

- Python 3.8 или выше
- Веб-камера (встроенная или внешняя)
- Windows, Linux или macOS

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
cd DMS/project
```


### 2. Установить зависимости

```
pip install -r requirements.txt
```

### 3. Скачать модель MediaPipe

Модель необходимо поместить в папку `models/`.

**Windows (PowerShell):**

```
Invoke-WebRequest -Uri "https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task" -OutFile "models/face_landmarker.task"
```

**Linux / macOS:**

```
curl -L "https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task" -o models/face_landmarker.task
```

## Использование

```
python main.py
```

### Управление

| Клавиша   | Действие                                                                                              |
| ---------------- | ------------------------------------------------------------------------------------------------------------- |
| `Q`            | Выход из программы                                                                            |
| `R`            | Сброс счетчиков и накопленной истории                                        |
| `S`            | Сохранить скриншот текущего кадра                                               |
| `+` / `=`    | Увеличить порог EAR (менее чувствительная детекция морганий) |
| `-` / `_`    | Уменьшить порог EAR (более чувствительная детекция)                  |
| `ПРОБЕЛ` | Приостановить / возобновить обработку                                        |

### Отображаемые параметры

| Параметр  | Описание                                                                                               |
| ----------------- | -------------------------------------------------------------------------------------------------------------- |
| **State**   | Общее состояние:`ATTENTIVE`, `WARNING`, `CRITICAL`                                         |
| **Eyes**    | Текущее состояние глаз:`OPEN` или `CLOSED`                                          |
| **EAR**     | Eye Aspect Ratio. Порог по умолчанию:`0.20`                                                  |
| **PERCLOS** | Доля времени с закрытыми глазами (история 60 кадров)                  |
| **Blinks**  | Общее число морганий и частота в минуту                                       |
| **Gaze**    | Предполагаемая зона внимания:`road`, `dashboard`, `mirror`, `lap`, `other` |
| **MAR**     | Mouth Aspect Ratio. Значения выше `0.6` — возможное зевание                     |
| **Pose**    | Углы головы:`Y` (поворот), `P` (наклон) в градусах                         |
| **FPS**     | Текущая частота кадров                                                                     |

## Метрики и пороговые значения

### PERCLOS (Percentage of Eye Closure)

Доля времени с закрытыми глазами за последние 60 кадров.

* **Расчёт:** `PERCLOS = число_закрытых_кадров / общее_число_кадров_в_истории`
* **Порог WARNING:** > 7.5%
* **Порог CRITICAL:** > 15%

### EAR (Eye Aspect Ratio)

Количественная мера открытия глаза.

* `EAR > 0.20` — глаз открыт
* `EAR ≤ 0.20` — глаз закрыт

Формула: `EAR = (|p2-p6| + |p3-p5|) / (2 * |p1-p4|)`

### MAR (Mouth Aspect Ratio)

Количественная мера открытия рта.

* `MAR > 0.6` в течение 15 кадров подряд — фиксируется зевок
* Частота зевков > 3 в минуту — excessive yawning

### Поза головы

| Угол | Описание                       | Критический порог |
| -------- | -------------------------------------- | --------------------------------- |
| Pitch    | Наклон вперёд/назад   | < -40° (microsleep)              |
| Yaw      | Поворот влево/вправо | > 45° (отвлечение)     |
| Roll     | Наклон к плечу             | > 30°                            |

### Зоны внимания

| Зона     | Pitch       | Yaw          | Пример                    |
| ------------ | ----------- | ------------ | ------------------------------- |
| road         | -20°…5°  | -25°…25°  | Взгляд на дорогу  |
| dashboard    | 5°…40°   | -20°…20°  | Приборная панель |
| left_mirror  | -20°…10° | 25°…50°   | Левое зеркало       |
| right_mirror | -20°…10° | -50°…-25° | Правое зеркало     |
| lap          | 40°…70°  | -20°…20°  | Телефон в руке      |

## Логирование

Все опасные события автоматически сохраняются в папку `logs/`.

**Файл лога:** `logs/session_YYYYMMDD_HHMMSS.json`

**Формат записи:**

```
{
  "session_id": "20260426_143052",
  "start_time": "2026-04-26T14:30:52",
  "events": [
    {
      "timestamp": "2026-04-26T14:31:15",
      "elapsed_seconds": 23.4,
      "type": "drowsiness",
      "severity": "critical",
      "message": "DROWSY: PERCLOS 53.3%",
      "details": {
        "perclos": 0.533
      }
    }
  ],
  "summary": {
    "duration_seconds": 120.5,
    "total_frames": 2410,
    "avg_fps": 20.0,
    "total_warnings": 5,
    "total_criticals": 2
  }
}
```

При каждом событии уровня `CRITICAL` автоматически сохраняется скриншот в `logs/screenshots/`.

## Конфигурация

Настройки по умолчанию заданы в `utils/config.py`. Их можно переопределить через JSON-файл:

```
from utils.config import Config

config = Config(
    camera_id=0,
    camera_width=640,
    camera_height=480,
    ear_threshold=0.20,
    perclos_threshold=0.15,
    alert_cooldown=3.0
)
config.save("my_config.json")
```

Запуск с пользовательской конфигурацией:

```
python main.py --config my_config.json
```

### Основные параметры

| Параметр      | По умолчанию | Описание                                                                                                    |
| --------------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `camera_id`         | 0                       | Индекс веб-камеры                                                                                    |
| `camera_width`      | 640                     | Ширина кадра                                                                                             |
| `camera_height`     | 480                     | Высота кадра                                                                                             |
| `ear_threshold`     | 0.20                    | Порог закрытия глаза                                                                              |
| `perclos_threshold` | 0.15                    | Порог сонливости по PERCLOS                                                                        |
| `alert_cooldown`    | 3.0                     | Минимальный интервал между одинаковыми предупреждениями (сек) |

## Ограничения

* Точность оценки позы головы зависит от положения камеры относительно лица
* Зоны взгляда могут требовать калибровки под конкретную посадку водителя
* Детекция зевания может давать ложные срабатывания при разговоре
